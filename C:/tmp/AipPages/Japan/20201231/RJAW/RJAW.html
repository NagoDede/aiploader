4. Lost communication procedures for arrival aircraft under radar navigational guidance
If radio communications with IWO Radar/Approach/GCA are lost for 1 minute in the pattern or 5 seconds(PAR)/15 seconds(ASR) on final approach, squawk Mode A/3 Code 7600 and;
(I)1.Contact IWO Tower
2.If unable, proceed in accordance with visual flight rules.
3.If unable, proceed TACAN IAF or IWOTO NDB at last assigned altitude or 3,000 feet whichever is higher, and execute instrument approach.
(II)Procedures other than above will be issued when situation required.

5. Automated Radar Terminal System (ARTS)
Aircraft flying under control of Iwo approach control in the approach area will be instructed to reply with discrete code Mode A/3 and Mode C.

If an aircraft with non-discrete code capability be instructed to reply with such code, it shall report a controller accordingly.
