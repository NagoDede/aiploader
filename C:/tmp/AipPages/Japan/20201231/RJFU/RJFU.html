2. Lost communication procedures for Arrival Aircraft under radar navigational guidance.
  If radio communications with NAGASAKI Approach/Radar are lost for 30 seconds, squawk Mode A/3 Code 7600 and :
(I)1.Contact NAGASAKI Tower.
2.If unable, proceed in accordance with Visual Flight Rules.
3.If unable, proceed to NAGASAKI VOR/DME at last assigned altitude or 4,000 feet whichever is higher, and execute instrument approach.
(II)Procedures other than above will be issued when situation required.

3. Trajectorized Airport Traffic Data Processing System (TAPS)

Aircraft flying under control of Nagasaki approach control in the approach control area will be instructed to reply with discrete code on Mode A/3 and Mode C.
If an aircraft with non-discrete capability be instructed to reply with the discrete code, it shall report a controller accordingly.
