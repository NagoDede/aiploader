3. Lost communication procedures for arrival aircraft under radar navigational guidance
If radio communications with HAMAMATSU Radar are lost for 1 minute in the pattern or 5 seconds(PAR)/ 15 seconds(ASR) on final approach, squawk Mode A/3 Code 7600 and ;
(I)1.Contact HAMAMATSU Radar/Tower.
2.If unable, proceed in accordance with Visual Flight Rules.
3.If unable, proceed to TACAN IAF at last assigned altitude or 3,000ft whichever higher, and execute instrument approach
(II)Procedures other than above will be issued when situation required.

4. Automated Radar Terminal System (ARTS)
Aircraft flying under control of Hamamatsu approach control in the approach control area will be instructed to reply with discrete code on Mode A/3 and Mode C.
If an aircraft with non-discrete code capability be instructed to reply with such code, it shall report a controller accordingly.
